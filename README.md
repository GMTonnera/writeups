## 📄 Formato dos Writeups

Cada writeup segue a estrutura abaixo:

```markdown
# Nome do Desafio / Room

- **Plataforma:** TryHackMe / HackTheBox / ...
- **Dificuldade:** Easy / Medium / Hard / Insane
- **Categoria:** Web / Pwn / Crypto / Forense / Misc / ...
- **Data:** YYYY-MM-DD

## Descrição
Breve resumo do desafio.

## Enumeração
Etapas de reconhecimento e coleta de informações.

## Exploração
Vulnerabilidades encontradas e como foram exploradas.

## Pós-exploração / Escalada de Privilégios
Técnicas utilizadas após o acesso inicial.

## Ferramentas Utilizadas
- `nmap`, `gobuster`, `burpsuite`, ...

## Referências
- Links úteis consultados durante a resolução.
```
